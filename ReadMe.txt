To set-up the lattepanda follow these steps 
 - Make sure the system date is in English-United States format (m/d/yyyy)
 - open the command line and execute venv.bat, this batch script will set up 
   the python environment and will update the project folder with the latest
   version avaliable from git
 - run "python GenConfig.py" and enter 'p' to set the production configuration
 - run "python Config.py"
